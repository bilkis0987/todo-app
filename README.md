## ğŸ› ï¸ Cara Menggunakan

### Instalasi

1. Clone atau download repository ini
2. Buka file `index.html` di browser Anda
3. Aplikasi siap digunakan

### Penggunaan

1. **Tambah Tugas**: Masukkan teks di kotak input dan klik tombol "Tambah" atau tekan Enter
2. **Tandai Selesai**: Klik tombol centang untuk menandai tugas sebagai selesai
3. **Edit Tugas**: Klik tombol edit untuk mengubah teks tugas
4. **Hapus Tugas**: Klik tombol hapus untuk menghapus tugas
5. **Statistik**: Secara otomatis menampilkan jumlah tugas
